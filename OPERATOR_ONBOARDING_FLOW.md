## ðŸ†˜ Common Issues & Solutions

### Issue: Driver Can't Access App
**Solution:**
- Check magic link expiration (30 days)
- Verify link was sent correctly
- Generate new link
- Test link yourself first

### Issue: Cases Not Appearing
**Solution:**
- Check driver reported correctly
- Verify vehicle is linked
- Check dashboard filters
- Check case status

### Issue: Webhook Not Working
**Solution:**
- Verify webhook URL is correct
- Check events are subscribed
- Test webhook endpoint
- Check webhook logs
